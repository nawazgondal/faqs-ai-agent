## 🚀 Quick Start (Copy & Paste)

### Prerequisites
```bash
# Download and install:
# - Python 3.9+ from https://python.org
# - Node.js 18+ from https://nodejs.org
# - Ollama from https://ollama.ai

# Pull Mistral model
ollama pull mistral
```

### Terminal 1 - Ollama
```bash
ollama serve
```

### Terminal 2 - Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Terminal 3 - Frontend
```bash
cd frontend
npm install
npm start
```

### Open Browser
```
http://localhost:4200
```

**Total setup time: ~15 minutes**
