## 💡 Example Usage

**User**: "How do neural networks work?"

**System**:
1. Embeds question → 384-D vector
2. Searches FAISS → Finds top-3 FAQs
3. Builds prompt → Adds FAQ context
4. Calls Mistral → Generates response
5. Returns → Answer + related FAQs

**Response**:
> "Neural networks are computing systems inspired by biological neural networks. They consist of interconnected nodes that work together to process information and learn patterns..."
>
> **Related FAQs:**
> - What is deep learning?
> - What is machine learning?
> - What is a neural network?

## 📊 System Architecture

```
USER
  ↓
ANGULAR CHAT UI (Port 4200)
  ↓ HTTP POST
FASTAPI BACKEND (Port 8000)
  ↓
SEMANTIC PIPELINE:
  1. Embed question (SentenceTransformer)
  2. Search FAISS index
  3. Get top-3 FAQs
  4. Build prompt context
  5. Call Ollama/Mistral
  6. Generate response
  ↓
DISPLAY RESULT
  - AI answer
  - Related FAQs
  - Show in chat
```
